Usage Guidelines

Button Groups

  • .btn-group - Horizontal group with 8px spacing between buttons
  • .btn-group-vertical - Vertical stack of buttons
  • .btn-group-connected - Buttons with merged borders (no gap)
  • .btn-group-sm / .btn-group-lg - Size modifiers for groups
  • .btn-group-mobile-stack - Forces vertical stacking on mobile

Button Toolbars

  • .btn-toolbar - Container for multiple button groups with 16px spacing
  • .btn-toolbar-justified - Space-between layout
  • .btn-toolbar-center - Centered layout
  • .btn-toolbar-right - Right-aligned layout
  • .btn-toolbar-divider - Visual separator between groups

Responsive Behavior

  • On mobile (< 768px), button groups wrap and allow buttons to flow to multiple lines
  • Toolbars stack vertically on mobile for better touch interaction
  • Connected button groups break apart on mobile with normal spacing
  • Toolbar dividers are hidden on mobile
  • All buttons maintain 44px minimum touch targets on mobile
