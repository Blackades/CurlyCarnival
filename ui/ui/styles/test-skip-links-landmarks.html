What Are Skip Links?

Skip links are hidden navigation links that become visible when focused with the keyboard. They allow keyboard users to bypass repetitive navigation and jump directly to main content areas.

Benefits of Skip Links

  • Saves time for keyboard users
  • Improves accessibility for screen reader users
  • Meets WCAG 2.1 Level A success criterion 2.4.1 (Bypass Blocks)
  • Enhances overall user experience

What Are Landmarks?

Landmarks are semantic regions of a page that help assistive technology users understand page structure and navigate efficiently. They can be created using HTML5 semantic elements or ARIA role attributes.

üîç Landmarks on This Page

  • Banner: The blue header at the top (role="banner")
  • Navigation: The gray sidebar on the left (role="navigation")
  • Main: This content area you're reading (role="main")
  • Contentinfo: The dark footer at the bottom (role="contentinfo")

How to Navigate by Landmarks

Screen reader users can navigate by landmarks using keyboard shortcuts:

  • NVDA: Press D to jump to next landmark
  • JAWS: Press R to jump to next landmark
  • VoiceOver: Press VO + U, then select "Landmarks"
